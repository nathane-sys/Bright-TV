# BrightTV Viewership Analytics

## Project Overview
This project analyzes BrightTV’s viewership data to uncover insights that can help increase subscription growth. The focus is on understanding user behavior, content performance, and engagement patterns to support Customer Value Management (CVM) strategies.

---

## Business Objective
The CEO of BrightTV aims to grow the company’s subscription base.  
This project provides data-driven insights to help:

- Improve user engagement  
- Optimize content strategy  
- Identify growth opportunities across regions and time  

---

## Dataset Description

The analysis is based on two datasets:

### 1. Viewership Data
- UserID  
- Channel  
- Watch timestamps  
- Duration of viewing sessions  

### 2. User Profile Data
- Age  
- Gender  
- Province  
- Customer demographics  

These datasets were joined using **UserID** to create a complete view of user behavior.

## Data Preparation

The raw data was processed using SQL:

- Converted timestamps from UTC to local time  
- Cleaned and standardized duration fields  
- Joined datasets (viewership + user profile)  
- Created new features:
  - Watch Hour  
  - Day of Week  
- Built a final analytical dataset for reporting  

---

## Key Analysis Performed

- Peak viewing hours analysis  
- Engagement by day of the week  
- Most viewed channels  
- Average watch time by channel  
- Regional engagement (province-level)  
- User behavior patterns  

---

## Key Insights

- Viewer activity peaks during afternoon and evening hours  
- Sports and live events drive the highest engagement  
- Gauteng and Western Cape lead in viewership  
- Weekends (especially Saturday) have the highest watch time  
- Some channels have high views but low retention  

---

## Recommendations

- Invest in live sports and event-based content  
- Schedule premium content during peak hours  
- Focus marketing on high-performing regions  
- Improve low-engagement channels  
- Use data-driven scheduling strategies
